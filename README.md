# 光・宇宙・政治 ニュースポータル

## ファイル構成

```
index.html   ← 閲覧者ページ（一般公開用）
admin.html   ← 管理者アップロードページ（管理者専用）
README.md    ← この説明書
```

---

## 使い方

### 【管理者】毎日の更新手順

1. `admin.html` をブラウザで開く
2. **ステップ1**：Anthropic APIキーを入力して「保存」
3. **ステップ2**：対象日付を選択（例：2026-04-21）
4. **ステップ3**：2つのPDFをアップロードして「AIで自動抽出する」をクリック
5. **ステップ5**：抽出結果を確認・URLを追加・必要なら削除
6. **ステップ4**：音声ファイル（MP3等）があればアップロード
7. **ステップ6**：「ブラウザに保存」で即時反映

### 【閲覧者】閲覧方法

- `index.html` をブラウザで開く
- カレンダーからデータのある日付をクリック
- 4つのタブで記事を切り替え
- PDFダウンロードボタンで元レポートを取得

---

## データ保存の仕組み

### ローカル（サーバー不要）モード
- 管理者が「ブラウザに保存」すると **localStorage** に保存
- **同じPC・同じブラウザ**で index.html を開くと自動反映
- ※ 異なるPC・ブラウザには共有されない

### サーバーホスティングモード（共有・公開向け）
JSON ファイルをサーバーに配置すると、誰でもアクセス可能になります。

```
（サーバールート）/
├── index.html
├── admin.html
├── data/
│   ├── index.json          ← 日付一覧
│   ├── 2026-04-21/
│   │   ├── index.json      ← ニュースデータ
│   │   ├── tech_report.pdf
│   │   ├── politics_report.pdf
│   │   └── audio.mp3
│   └── 2026-04-22/
│       └── index.json
```

**data/index.json の形式：**
```json
{
  "dates": ["2026-04-20", "2026-04-21", "2026-04-22"]
}
```

**data/YYYY-MM-DD/index.json の形式：**
`admin.html` の「JSONファイルとしてダウンロード」で生成されます。

## ホスティング方法

### GitHub Pages（無料・推奨）
1. GitHubリポジトリを作成
2. ファイルをアップロード
3. Settings → Pages → Branch: main で公開

### Netlify（無料）
1. netlify.com でアカウント作成
2. フォルダをドラッグ&ドロップでデプロイ

### 自社サーバー
静的ファイルとしてそのまま配置可能（Apache/Nginx等）

---

## 注意事項

- Anthropic APIキーは管理者のみが知る状態を保ってください
- admin.html はパスワード保護するか、ローカルのみで使用を推奨
- 音声ファイルをBase64でlocalStorageに保存する場合、ファイルサイズに制限あり（5MB程度まで推奨）
- サーバーモードでは音声ファイルもサーバーに配置し、JSONのURLを実際のパスに変更してください
